SITCON Board 是固定對應 `sitcon-tw/2027` 的籌備工作看板。使用者透過 GitLab OAuth 登入、確認主要組別，並在同一個 Board 上快速開卡、移動、指派、調整期限與關閉。

## Start here

使用 [getting started](/docs/guides/getting-started/) 啟動 PostgreSQL、backend 與 web。跨越 package boundary 前先讀 [architecture](/docs/reference/architecture/)；部署前逐項核對 [configuration](/docs/reference/configuration/)。

## Sources of truth

| Concern                | Source                                   |
| ---------------------- | ---------------------------------------- |
| HTTP contract          | `api/**/*.tsp`                           |
| Database schema        | `server/db/migrations`                   |
| Backend boundaries     | `ARCHITECTURE.md` and architecture tests |
| Browser tokens         | `packages/ui/src/styles/tokens.css`      |
| Visual rules           | `design.md`                              |
| Cross-project commands | `Justfile`                               |

Generated OpenAPI 與 TypeScript declarations 是 review artifacts，不是手動編輯面。修改 TypeSpec 後執行 `pnpm generate` 並提交全部輸出。
